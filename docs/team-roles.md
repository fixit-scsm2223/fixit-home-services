# Team Roles

## Group A - Customer Marketplace Module

Group A handles the customer workflow:
customer searches providers, books service, tracks job, confirms cost, and reviews.

### Member 1 - Customer UI Lead

- Home page
- Service categories page
- Provider listing page
- Provider details page

### Member 2 - Customer Booking UI Lead

- Create booking page
- Customer dashboard
- My bookings page
- Job ticket page
- Review page

### Member 3 - Customer API Lead

- Browse providers API
- Create job API
- Customer job history API
- Review API

### Member 4 - Auth, Customer Database & Security Lead

- User table
- Login/register
- JWT
- Role protection
- Job and review database queries

## Group B - Provider/Admin Operations Module

Group B handles the provider and admin workflow:
provider applies, admin verifies, provider receives jobs, updates status, and completes work.

### Member 5 - Provider Module Lead

- Provider profile page
- Provider dashboard
- Provider category selection
- KYC upload page
- Provider job request page

### Member 6 - Admin Module Lead

- Admin dashboard
- Provider verification page
- Manage categories
- View providers
- View jobs

### Member 7 - Provider/Admin API Lead

- Provider profile APIs
- KYC upload API
- Admin verification APIs
- Provider job status APIs
- Category management APIs
