# FixIt Project Scope

FixIt is a cross-platform local home services marketplace for booking verified local service providers.

## Target users

1. Customer
2. Service Provider
3. Administrator

## Core features

- User registration and login
- JWT authentication
- Role-based access
- Service category catalogue
- Provider profile
- Mock KYC document upload
- Admin provider verification
- Customer booking workflow
- Job ticket with status updates
- Final cost confirmation
- Review and rating
- Responsive UI
- Capacitor Android build

## Main workflow

1. Provider registers and creates a profile.
2. Provider uploads mock KYC document.
3. Admin verifies the provider.
4. Customer browses verified providers.
5. Customer books a service.
6. Provider accepts the job.
7. Provider updates job status.
8. Customer confirms final cost.
9. Customer leaves a review.
